# Library Frontend

图书馆管理系统前端项目，使用React 19.2.4 + Redux + React Router 7.0开发。

## 功能特性

- 用户认证（登录、注册、登出）
- 书籍管理（查看、添加、编辑、删除）
- 用户管理（查看、权限修改）
- 借阅流通（借书、还书、历史记录）
- 管理员功能（日志查看）

## 技术栈

- React 19.2.4
- React Router 7.0
- Redux Toolkit
- Axios
- 原生CSS

## 安装与运行

1. 安装依赖：
   ```bash
   npm install
   ```

2. 启动开发服务器：
   ```bash
   npm start
   ```

3. 构建生产版本：
   ```bash
   npm run build
   ```
