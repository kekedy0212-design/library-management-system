# 图书馆管理系统 API 接口文档 (v1.0.0)

> **Base URL**: `http://localhost:8000/api/v1`  
> **认证方式**: Bearer Token (JWT)  
> **Content-Type**: `application/json` (登录接口使用 `application/x-www-form-urlencoded`)  
> **Swagger UI**: `http://localhost:8000/docs`

---

## 1. 认证模块 (Authentication)

### 1.1 注册

**POST** `/auth/register`

**请求体 (JSON)**:
```json
{
  "username": "string",
  "email": "user@example.com",
  "password": "string"
}
```

**响应 (201 Created)**:
```json
{
  "username": "string",
  "email": "user@example.com",
  "id": 0,
  "role": "reader",
  "is_active": true
}
```

**错误码**:
- `400` - 用户名或邮箱已存在
- `422` - 请求体格式错误

---

### 1.2 登录

**POST** `/auth/login`

**请求体 (form-data)**:
| 参数 | 类型 | 必填 | 描述 |
| :--- | :--- | :--- | :--- |
| username | string | ✅ | 用户名 |
| password | string | ✅ | 密码 |

**响应 (200 OK)**:
```json
{
  "access_token": "eyJhbGciOiJIUzI1NiIs...",
  "token_type": "bearer"
}
```

**错误码**:
- `401` - 用户名或密码错误
- `400` - 账号未激活

---

### 1.3 登出

**POST** `/auth/logout`

**响应 (200 OK)**:
```json
{
  "msg": "Successfully logged out"
}
```
> 后端无状态，前端只需清除本地存储的 Token。

---

## 2. 书籍管理 (Books)

### 2.1 搜索/浏览书籍

**GET** `/books`

**查询参数**:
| 参数 | 类型 | 必填 | 描述 |
| :--- | :--- | :--- | :--- |
| q | string | ❌ | 搜索关键词（标题/作者/ISBN） |
| skip | integer | ❌ | 分页偏移量，默认 0 |
| limit | integer | ❌ | 每页数量，默认 100 |

**响应 (200 OK)**:
```json
[
  {
    "id": 1,
    "title": "Clean Code",
    "author": "Robert C. Martin",
    "isbn": "9780132350884",
    "description": "A handbook of agile software craftsmanship.",
    "total_copies": 5,
    "available_copies": 3,
    "location": "3F-计算机区-A排",
    "created_at": "2026-04-13T10:00:00Z",
    "updated_at": null
  }
]
```

---

### 2.2 获取单本书籍详情

**GET** `/books/{book_id}`

**响应 (200 OK)**:
```json
{
  "id": 1,
  "title": "Clean Code",
  "author": "Robert C. Martin",
  "isbn": "9780132350884",
  "description": "...",
  "total_copies": 5,
  "available_copies": 3,
  "location": "3F-计算机区-A排",
  "created_at": "2026-04-13T10:00:00Z",
  "updated_at": null
}
```

**错误码**:
- `404` - 书籍不存在

---

### 2.3 新增书籍 (🔒 需管理员/图书管理员权限)

**POST** `/books`

**请求体 (JSON)**:
```json
{
  "title": "string",
  "author": "string",
  "isbn": "string",
  "description": "string (optional)",
  "total_copies": 1,
  "location": "string (optional)",
  "available_copies": 1  // 可选，默认等于 total_copies
}
```

**响应 (201 Created)**:
返回创建的完整书籍对象（同上）。

**错误码**:
- `401` - 未认证
- `403` - 权限不足
- `400` - ISBN 已存在

---

### 2.4 修改书籍信息 (🔒 需管理员/图书管理员权限)

**PUT** `/books/{book_id}`

**请求体 (JSON)** (所有字段可选):
```json
{
  "title": "string",
  "author": "string",
  "isbn": "string",
  "description": "string",
  "total_copies": 10,
  "available_copies": 8,
  "location": "string"
}
```

**响应 (200 OK)**:
返回更新后的完整书籍对象。

**错误码**:
- `401` / `403` / `404`

---

### 2.5 删除书籍 (🔒 需管理员/图书管理员权限)

**DELETE** `/books/{book_id}`

**响应 (200 OK)**:
```json
{
  "msg": "Book deleted successfully"
}
```

## 3. 用户管理 (Users) 🔒

> 以下接口均需认证，部分需管理员权限。

### 3.1 获取用户列表 (需管理员/图书管理员)

**GET** `/users`

**查询参数**:
| 参数 | 类型 | 描述 |
| :--- | :--- | :--- |
| skip | integer | 分页偏移量，默认 0 |
| limit | integer | 每页数量，默认 100 |

**响应 (200 OK)**:
```json
[
  {
    "id": 1,
    "username": "admin",
    "email": "admin@library.com",
    "role": "admin",
    "is_active": true
  }
]
```

---

### 3.2 修改用户信息 (需系统管理员)

**PUT** `/users/{user_id}`

**请求体 (JSON)** (可修改启用状态或角色):
```json
{
  "is_active": false,
  "role": "librarian"
}
```

**响应 (200 OK)**:
返回更新后的用户对象。

**特殊限制**:
- 不能修改自己的角色
- 只有 `admin` 可以修改 `role` 字段

**错误码**:
- `403` - 权限不足
- `404` - 用户不存在

---

### 3.3 重置用户密码 (需管理员/图书管理员)

**POST** `/users/{user_id}/reset-password`

**查询参数**:
| 参数 | 类型 | 描述 |
| :--- | :--- | :--- |
| new_password | string | 新密码 |

**响应 (200 OK)**:
```json
{
  "msg": "Password reset successfully"
}
```

---

## 4. 借阅与归还 (Borrow/Return)

### 4.1 发起借书请求 (🔒 需登录)

**POST** `/borrow-requests`

**请求体 (JSON)**:
```json
{
  "book_id": 1
}
```

**响应 (200 OK)**:
```json
{
  "id": 100,
  "user_id": 5,
  "book_id": 1,
  "request_date": "2026-04-13T12:00:00Z",
  "status": "pending",
  "user": { ... },
  "book": { ... }
}
```

**错误码**:
- `400` - 书籍库存不足或已存在未完成的请求

---

### 4.2 发起还书请求 (🔒 需登录)

**POST** `/return-requests`

**请求体 (JSON)**:
```json
{
  "borrow_record_id": 100
}
```

**响应 (200 OK)**:
```json
{
  "id": 101,
  "status": "return_pending",
  ...
}
```

**错误码**:
- `400` - 借阅记录无效或状态不允许还书

---

### 4.3 查看个人借阅历史 (🔒 需登录)

**GET** `/users/me/borrow-history`

**响应 (200 OK)**:
返回当前用户的所有借阅记录数组。

---

### 4.4 获取待审批请求 (🔒 需管理员/图书管理员)

**GET** `/requests/pending`

**响应 (200 OK)**:
返回所有状态为 `pending` 或 `return_pending` 的请求数组。

---

### 4.5 审批请求 (🔒 需管理员/图书管理员)

**PUT** `/requests/{request_id}/process`

**请求体 (JSON)**:
```json
{
  "action": "approve",  // 或 "reject"
  "notes": "string (optional)"
}
```

**响应 (200 OK)**:
返回更新后的借阅记录。

**业务逻辑**:
- 批准借书：库存 `available_copies` 减 1，状态变为 `approved`，自动生成应还日期（借阅期 14 天）。
- 批准还书：库存 `available_copies` 加 1，状态变为 `returned`。
- 拒绝请求：状态变为 `rejected`，库存不变。

**错误码**:
- `400` - 请求状态非待审批，或批准借书时库存不足

## 5. 管理员专属 (Admin)

**GET** `/admin/logs`

**查询参数**:
| 参数 | 类型 | 描述 |
| :--- | :--- | :--- |

```json
  "lines": 20,
  "logs": [

    "2026-04-13 10:01:00 [WARNING] ..."
  ]
}
```

## 6. 通用说明

### 认证 Header 格式
所有需要认证的接口，请求头需携带：
```
Authorization: Bearer {access_token}
```

### 角色与权限速查表

| 角色 | 权限范围 |
| :--- | :--- |
| **reader** | 搜索/查看书籍、借/还书申请、查看个人历史 |
| **librarian** | 以上全部 + 管理书籍、管理用户状态、审批借阅 |
| **admin** | 以上全部 + 修改用户角色、查看系统日志 |

### 常见 HTTP 状态码

| 状态码 | 含义 |
| :--- | :--- |
| 200 | 成功 |
| 201 | 创建成功 |
| 400 | 请求参数错误（业务逻辑拒绝） |
| 401 | 未认证（Token 无效或缺失） |
| 403 | 权限不足（角色不满足） |
| 404 | 资源不存在 |
| 422 | 请求体验证失败（字段格式错误） |
| 500 | 服务器内部错误（查看后端日志） |

**前端对接注意事项**:
- 登录接口使用 **form-urlencoded**，非 JSON。
- Token 默认有效期 30 分钟，建议前端在 401 时自动跳转登录。
- 分页参数 `skip` 和 `limit` 均为可选。
- 图书位置 `location` 为可选字段，可用于前端展示书架信息。
